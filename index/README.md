## ファイル構成

```
/
├── README.md
├── index/
|   └── README.md                    ←本資料。どこにどのような文書があるかが記載されています。
├── guidelines/                    ←各種手順書、運用ルールが記載された文書を格納するディレクトリです。
|   └── setup.md                    ←初回勉強会参加の前に、行うべき環境構築手順などが記載されています。
└── meetings/                    ←各勉強会毎の資料を格納するディレクトリです。
    ├── 第0回/                    ←各勉強会のテンプレートです。コピペして使用してください。
    | └── README.md               ←勉強会の記録を残す文書です（演習問題など、適宜ディレクトリやファイルを分ける）
    ├── 第1回/                    ←第1回の勉強会のディレクトリです。第2回以降も順次作成していきます。
    | └── README.md
    …
```
